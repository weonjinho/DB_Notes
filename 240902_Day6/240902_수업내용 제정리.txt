240902_수업내용 제정리

문제를 분석하는 순서 :
1. 대상 릴레이션을 먼저 찾아낸다.
2. 컬럼에 조건이 필요한지 판단한다. 
3. 조건을 가지고 튜플을 선정한다.
4. 기능(사용할 함수)을 판단한다.

select 절 : 회면에 출력하고자 하는 부분을 선정하는 부분. ( 어떤 컬럼을 출력할갓인지를 알려주는 역할, 출력하는게 아니다. )
from 절 : 대상 테이블
where 절 : 튜플 선정을 하기 위한 조건.
order by 절 : 정렬

조건식 : 참과 거짓을 구분하기 위해 만들어진 식.
하나의 튜플은 여러개의 컬럼으로 구성되었다.
튜플을 선정할때 특정 컬럼의 값으로 선정하겠다.

집계함수 특징 : 출력한 결과가 1개만 나온다.

함수의 정의 : 함수는 특정 기능을 기지고 있고, 함수는 리턴값을 기지고 있다.
설명 예제: count(*) : 모든 튜플의 갯수를 counting 한 결과를 return 해준다.

select id from member;
대상 테이블은 member이다, 회면에 출력괼것은 id이다.

실행 순서 :
	from
	where
	group by
	having
	select
	order by

테이블 분석 순서 :
	테이블 ---> 컬럼 ---> 속성 ---> 제약조건

함수 : 호출 ---> 매개변수를 통해 값을 넘겨준다 ---> 함수 실행 ---> 결과값 return.
